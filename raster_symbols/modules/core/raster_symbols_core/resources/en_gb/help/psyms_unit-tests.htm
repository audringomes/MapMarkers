Unit tests

The munit tests for the raster_symbols software is organised in a separate product raster_symbols_test.

To run all tests, follow the steps below:

1.

Add the product munit to your session;

2.

Add the product raster_symbols_test to your session;

3.

Load the test modules, either with the file load.magik in the product directory or by selecting and loading of the module raster_symbols_test in the Smallworld Module Manager;

4.

Run the tests by loading the file run.magik in the product directory.

The results will be written to the directory specified in the environment variable MUNIT_OUTPUT_DIR, and when not defined, the folder results in the test product.

 

The following tests are defined:

1. Creation basics

1a. Creation of a new symbol
1b. Creation of a symbol from a definition
1c. Modifying the default symbol

 
2. XML manipulations

2a. Create XML representations of symbols and symbol properties
