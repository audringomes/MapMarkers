Configuration files

Raster Symbols, Raster Symbol Definitions, Goto Mappings and options can be saved in an xml-file. The Management GUI has a button Save. (see Save) and Save As...

These files can be loaded by:
 

•

Plugin Autoload

•

Configuration GUI

•

Magik
