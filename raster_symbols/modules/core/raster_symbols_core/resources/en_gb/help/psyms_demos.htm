Demo's

Associated Pictures

 

1.

Start a session that contains the Related Documents from Common Office (module cs_base_model), for example Electric Office or Water Office.

 

2.

Load the module raster_symbols (you can use load_raster_symbols.magik)

 

3.

Load the module rs_associated_pictures

 

»

This will add the symbol_definition cs_ref_template and the code for a special subclass of the drawing method draw_raster_symbol()

 

4.

Start a suitable graphics application

 

5.

If you did not configured the raster_symbols plugin for the application app, you can add it dynamically with !add_rsp!(app) from the load_raster_symbols.magik.

 

»

You can start the Configuration GUI with !smi!()

 

6.

Add your own image file to the Related Documents, or use an existing bitmap;

 

7.

Add the drawing method draw_raster_symbol() to the style records of the cs_document_reference geometries;

You can do this with the RWO Style Mappings Tab or in the default Style Application.

 

8.

Place a trail point on or near the terrain object that is related to the picture;

 

9.

Open the Manage External Files Dialog, select a picture and select Create File Symbol
 

 

»

The moment the geometry is drawn, it will check for a thumbnail_raster_symbol for this object. If this is found, it will draw the Thumbnail raster Symbol of the picture. If not, it will first create the symbol;
If this raster is selected, the actual raster is displayed in the map.

Value Mapping

This demo is based on Water Office. See also the video on the website.

 

1.

Create and run a query for all Measurement Points;

 

2.

Select the records in the result of the Explorer and Click Goto;

 

»

If there is no Generic or specific Mapping, the points are show with the default symbol and default text;

 

3.

If one or more records are not displayed, check View Defaults Goto size and raise its value;

 

4.

Open the Configuration GUI

 

5.

Select the Symbol Definition "Bar-BlueRed" and Click Create Symbol

 

6.

Select the realised Symbol "Bar-BlueRed"

 

7.

Open the tab Goto Mappings and select in the mapping tree Specific

 

8.

Select in the Rwo Hierarchy List the object Measurement Point

 

9.

Click the button Set

 

10.

Refresh the map

 

»

Measurement points are now drawn with a bar graph.

 

 

Changing the properties of the symbol:

 

11.

Select in the Configuration Gui the realised Symbol "Bar-BlueRed" and the tab Symbol Properties

 

12.

Add the value_selector outlet_pressure_level, which is actually a field

 

13.

Define (optional) a value scale.

 

14.

Refresh the map

 

»

The height of the bar represents the value returned by the value_selector.
