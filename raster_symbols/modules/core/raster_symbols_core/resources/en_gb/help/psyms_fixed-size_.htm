Fixed_size?

The property fixed_size? determines if a raster symbol is drawn fixed size on the screen (value = "Yes"), or with a fixed width relative to the geographic world. When fixed_size? is "No", the value of draw_size_mm is used and should be present.

The draw_size_mm should be specified in world units (mm).

 

Fixed Size = fixed pixel size at screen:

1:1000

1:2400

 

Not fixed size = fixed size in world units:

1:1000

1:2400
