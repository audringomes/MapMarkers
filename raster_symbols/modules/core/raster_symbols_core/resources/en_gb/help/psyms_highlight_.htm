Highlight?

The option Highlight? changes the Draw that is initiated by pressing the button Draw.

The raster_images are not different when a symbol is 'drawn higlighted'. In stead, the bounds of the raster are drawn in the application highlight style, and any text is drawn using the style stored in property highlight_text_style.
