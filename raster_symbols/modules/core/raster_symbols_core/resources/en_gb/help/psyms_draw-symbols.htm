Operation

The Drawing Speed Monitor samples drawing speed of normal gis_style symbols and (when toggled) raster symbols. Pressing the Refresh button will refresh the main map view and show the speed of drawing of all symbols in the map. It takes only the first 100 to avoid endless drawing.

Click to enlarge

In the example above 62 Point Geometries are drawn with an average of 0.2245 ms.

When the toggle DrawPSP? is set, the drawing speed of the raster symbols with the name selected in the drop down box is shown.
