Symbol (definition) list

The Symbol (definition) list contains a list of available symbols and symbol definitions. The selected symbol is used in the creation of GOTO mappings and the generation of custom draw methods.

 
The image of a Raster Symbol Definition has a colour that reflect the status of the raster Definition:

 

A non-realised raster Definition

 

A realised raster Definition

 

A raster Definition that raised an error on realisation.

 

Operations:

Select

The properties of the selected symbol (definition) are displayed in the raster Symbol Properties List. The symbol is drawn at the Preview canvas. When authorised, the symbol can be renamed by a Double Click.
 

Double Click

A double click on an unrealised raster definition will raise a question if the symbol should be realised. (Only if this isn't done yet.) In all other cases it will enable modifying the name.

 

Realise

Creates a raster symbol from the selected definition.
 

 Delete

Deletes the selected raster(definition).

 

Draw

Draws the selected raster(definition). See Draw.

 
The popup menu contains the following actions:

Copy

Make a copy of the selected raster(definition).

Show in Explorer...

Select in the Windows explorer the raster image file.

Show Symbols only?

When selected, the list will show only (realised) raster symbols and no raster definitions.
