String_selector

The property :string_selector specifies a method (or field) that should be used to get the text string that is to be drawn.

The default raster has the value :rs_string. This is implemented as follows:

 

*cb**gis ws_example_open* 

 *** The 1 basic or debug method containing the string "rs_string".

rs_string  IN  sw:rwo_record    B (subclassable and redefinable)


        ## Default string method for a raster_symbol that returns the
        ## value of method :tooltip_description or when not defined, the
        ## value of method :description()

 

If sending the string_selector to a record returns an error, the text "ERROR" is drawn.
