Features

These are the major features of MapMarkers:

Use of png files and full transparency

The object colour in Smallworld 5 has an additional value alpha for specifying transparency. This allow the use of transparency. Because it uses alpha channel transparency, there is no limit on the design or the creation of raster symbols.

Raster symbols are created from common raster format files. The full support for partial or alpha channel transparency allows endless possibilities in the design of the symbols. I also supports the handling of photographs or grid files. See some examples below.

Enhanced GOTO

The MapMarkers can be configured to be used for 'GOTO symbols'. The system will draw a symbol with (optional) text, that identifies the object clearly.

- An mapping can be defined generic on geometry category (:point, :linear, :text, :area) or for specific geometries.

- When at the symbol level goto styles (:area and :line) are define, they can be used to draw the geometries. This is optional.

- A 'sticky mode' will keep on drawing, allowing the user to zoom and keep the object in sight.

Fixed size

The usual drawing is absolute, relative to the screen. In case of raster symbols this is "Fixed size". When the symbol is drawn not fixed size, the raster symbol object should contain a value of width in world mm. In this case the symbol will be visually enlarged when zoomed in, and vanish when zoomed out.

Useful in Thematic Mapping

 

Raster symbols can be used in the style definitions of the Thematic Mapping plugin.

Text drawing

All raster symbols can draw a text on a specified location relative to the pixel area. When a geometry of type text is drawn, it will draw it's value. The :string_selector property specifies what text should be drawn. A default implementation of the method :rs_string is provided and can be subclassed.

For every raster symbol, a text style can be defined.

Values and graphs

When a :value_selector is defined, a raster symbol can display this value as a graph. A basic statistical engine calculates range and scale, which can be superseded by passing the valuescale as an argument.

Configuration Interface

To ease the deployment of Raster Symbols, a GUI is provided that allows the creation, modification and usage of the Raster Symbols. But MapMarkers does not need this interface to work.

Highly configurable (XML)

This version of the MapMarkers does not store data in a (Smallworld) database. Storage is in xml and/or Magik. The design is such that every object can be mapped to a table. For example, the GOTO mappings are stored as (subclasses) of the object goto_mapping.

Fast

The speed of drawing of pure raster type symbols is high. When carefully designed, raster symbols can be used without loss of drawing speed. (See also Speed)
To test drawing speed you can use the Drawing Speed Monitor.
