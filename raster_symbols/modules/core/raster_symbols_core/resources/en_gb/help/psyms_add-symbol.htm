Add symbol

The function Add symbol allows the creation of a raster Symbol from a selected raster image file.
 

•

A file dialog opens with the directory of the bitmap resources of te current module (see Modules)

Currently only png files can be selected.

•

If you select a file in another folder, and the module isn't known to the handler yet, it will add the module to the list.

•

If you select a file in another folder and this folder isn't a bitmap resource for a module, an error is raised.

•

A raster raster symbol will be created with defaults as described in Properties.

•

The Symbol (definition) list will be updated to show the new raster Symbol.
