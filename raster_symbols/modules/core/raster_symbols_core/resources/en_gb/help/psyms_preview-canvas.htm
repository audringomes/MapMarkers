Preview canvas

The Preview Canvas is used when a realised raster Symbol is selected. The pixels from the raster_image are mapped 1:1 to the canvas pixels and centered.

When a raster is too large to be drawn, an error message is drawn at the canvas.

Highlight

When the GUI option Highlight? is set, the canvas shows a pixel box in highlight style as well as the text_highlight_style of the symbol.
