Raster Symbol Definitions

Raster Symbols can be defined in a simple prototype object. Actually a property_list. Such a Symbol Definition can be converted to a real Raster Symbol anytime. This enables realisation when needed.

Symbol Definitions are, like the realised Raster Symbols handled by the raster_symbol_handler.
A Symbol Definition is created with the method add_symbol_definition("name",properties), and realised by the method create_symbol_from_definition("name").

Symbol definitions can be stored in and retrieved from the Raster Symbol Configuration files.

They show up in the GUI with a coloured circle. An popup option can hide them.
The module raster_symbols constains a file in which a couple of definitions are created:


_block
        _local l_props << property_list.new()
        l_props[:module_name] << :raster_symbols
        l_props[:filename] << "house_orange.png"
        l_props[:type] << :raster
        l_props[:pixel_offset] << pixel_coordinate.new(-13,0)
        l_props[:draw_text_string?] << _true
        l_props[:text_raster_justification] << :top_centre
        l_props[:text_justification] << :bottom_centre
        l_props[:string_selector] << :rs_string
        raster_symbol_handler.add_symbol_definition("House",l_props)
_endblock
