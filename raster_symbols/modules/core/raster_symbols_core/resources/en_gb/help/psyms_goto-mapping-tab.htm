Goto Mappings Tab

The Goto Mapping tab is used to create and modify Goto Mappings for three different types: Generic, Specialised and Specific.

 

Create Generic mapping

1.

Select the tab Goto Mappings.

2.

Select a Symbol in the Symbol (definition) list.

3.

Select the line with the geom type you want to configure.

4.

Press Set

5.

Set the option to use the goto_style from the selected Symbol.

» All point geometries in a goto request will use the selected symbol.

Setting the option use_style? has no effect for geometries of type :point and :text

Create Specialised mapping

1.

Select the tab Goto Mappings.

2.

Select a Symbol in the Symbol (definition) list.

3.

Select the line with the special type (e.g. Numerator) you want to confiure

4.

Press Set

» If the Numerator is selected, all goto requests with an ordered_geometry_set are drawn with the selected symbol.

Other types may have different behaviour, depending on your configuration. (See Handling of mappings.)

Create Specific mapping

Specific mappings are mappings for specific geometries.
 

1.

Select the tab Goto Mappings.

2.

Select a Symbol in the Symbol (definition) list.

3.

Select the heading Specific

4.

Select in the rwo hierarchy list the element you want to configure with the selected symbol. The selection can be one or more geometries or a collection. If you specify a collection, the mapping will show all, meaning all geometries of the collection should be drawn with the selected raster Symbol.

5.

Press Set

 

» A new mapping is created for the selection.

6.

(optional) Set in the mapping the option use? to use the goto_style from the selected Symbol.

7.

(optional) Set in the mapping the option mapped geometries? to map all mapped geometries to use this goto mapping. If the parent geometry field of a mapped geometry is selected, this is filled in automatically with True

Modify Generic Mapping

1.

Select the raster symbol you want to use

2.

Select beneath the heading Generic? the line with the mapping you want to modify

3.

Press Set

4.

Set the option use? the goto_style from the selected Symbol.

»

The mapping is changed

Modify Specialised mapping

1.

Select the raster symbol you want to use

2.

Select beneath the heading Specialised? the line with the mapping you want to change

3.

Press Set

»

The mapping is changed
