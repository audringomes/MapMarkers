Highlight_text_style

The Highlight Text Style is used when a raster symbol is drawn with the parameter highlight? is true, which is done for example in the standard custom_draw_method that is generated. This can be previewed in the GUI (see Highlight?)

 
Possible values for the symbol property highlight_text_style are:

Value

Action

_unset

the style is set from the default highlight_text_style

a text_style

the text_style is used

"default"

the default highlight_text_style is used

"derived"

the default highlight_text_style is copied with the colour value from the application.highlight_colour
