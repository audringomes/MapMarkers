MapMarkers

This software is a spin-off of a Proof of Concept for drawing normal bitmaps on the maps in Smallworld GIS. The investigation revealed easy use of bitmap resources with transparency, mainly caused by the new alpha property of the object 'colour'.
The use of grids and colour mappings for value mapping was a logical start, and is still a part of the module set.
The Configuration GUI is written as subclass of model, the plugin has actions and a toolbar. It seems to be very logical to build the GUI in the Style Designer application. There is no database involved, except for the tab RWO mappings, which interacts with the style view.
The software is shared with the whole Smallworld Community to be an inspiration. The GUI is mainly based on legacy widgets, but all of them function as designed.
The software will work from 4.3.0 and is lately tested on 5.2.2.

The software is public domain, with courtesy of Realworld Software Products.

The basic idea behind the MapMarkers is to improve the visual interaction between the map and the user.
A better map makes it easier and faster to find the things a user needs. It displays and emphasize what is important for his specific task.
In the real world there are many users with many specific tasks.

Therefore, this product allows you to create configurations for different tasks.
