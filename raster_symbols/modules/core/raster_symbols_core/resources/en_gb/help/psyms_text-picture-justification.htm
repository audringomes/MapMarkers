Text_raster_justification

The text_raster_justification specifies the position of the text relative to the raster. Besides the standard position values, there are two extra positioning possibilities: :text_coordinate and :text_box

Below three examples, two for a pushpin with 64x64 pixels, and one for a Text Callout symbol.

 

Standard positions

:top_left

:top_centre

:top_right

:centre_left

:centre_centre

:centre_right

:bottom_left

:bottom_centre

:bottom_right

Text Coordinate

 

 

 
a pixel_coordinate  (in this example pixel_coordinate.new(47,66))

 

 

a pixel_bounding_box (in this example with values: xmin:2, ymin:30, xmax:63, ymax:63)

Text will be drawn within the limit of this box.
