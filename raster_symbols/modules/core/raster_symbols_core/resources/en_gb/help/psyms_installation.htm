Installation

The installation is very simple. Below are the steps for manual load.

1.

 

Start the Module Manager

2.

 

Select Add Product

3.

 

Select in the File Open dialog the product.def of the Raster Symbol product

»

The product is loaded and a list of modules is displayed

4.

 

Select the module raster_symbols_gui

5.

 

Press Load Module

»

 

The module raster_symbols_core and raster_symbols_gui are loaded

It is possible to load the module raster_symbols_core only.

See Plugin configuration on how to modify your configuration files to add the raster_symbol_plugin and optionally the toolbar.

For installation and configuration of ImageMagick see Image Magick.
