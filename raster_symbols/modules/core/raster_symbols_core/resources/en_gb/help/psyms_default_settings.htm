Default Settings

When symbols are created, and the value for some properties isn't provided, the following methods for default values are used.

default_goto_area_style  IN  sw:raster_symbol    B


        ## The default GOTO area style. A fill_style that is created
        ## the first time the method is called. The method will be
        ## converted automatically in a shared_constant


default_goto_line_style  IN  sw:raster_symbol    B


        ## The default GOTO line style. Build and cached as a shared
        ## constant with the same name


default_highlight_style  IN  sw:raster_symbol    B


        ## The highlight linestyle. A line style with colour derived from
        ## application.highlight_colour.


default_highlight_text_style  IN  sw:raster_symbol    B


        ## The default sw:text_style for drawing text in highlight


default_measurement_text_style  IN  sw:raster_symbol    B


        ## The default sw:text_style for drawing the used value scale and
        ## value method when drawing values.


default_text_parameters  IN  sw:raster_symbol    B


        ## The default value for text parameters
        ## :text_justification            (:bottom_left)
        ## :text_raster_justification    (:top_right)
        ## :text_scale_factor             (1.0)


default_text_style  IN  sw:raster_symbol    B


        ## The default sw:text_style for drawing text

All default values are shown in the tab Default Settings of the Configuration GUI.
