Document conventions

The following typographic conventions are used throughout the document:

 

 

•

Arial Bold face is used for the names of menus, editors and dialog boxes; also for the names of menu commands, buttons and fields in editors and dialog boxes.

 

•

Italic face is used:

 

 

• for emphasis

 

 

• in cross-references to sections, figures and tables, and titles of documents

 

 

• in References to module names or part of products

 

•

Verdana Bold Blue is used for the names of database Tables/Objects.

 

•

Verdana Bold Blue Italic is used for the names of database fields.

 

•

ALL CAPS are used for Keyboard keys, for example CTRL.

 

•

Text that represent a file path, e.g. c:\smallworld is marked 8pt bold

 

•

Calibri Green Bold is used for the names of code object references in the text.

 

•

Examples of source code without named syntax highlighting are given in courier green, for example:
