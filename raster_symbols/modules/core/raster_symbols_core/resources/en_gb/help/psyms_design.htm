Design

The basic object is raster_symbol, which has two subclasses: grid_raster_symbol and thumbnail_raster_symbol. The manipulation of all of them is done by the raster_symbol_handler. There exists only one instance of a particular symbol with a certain name. When properties are changed (e.g. style) a copy of that property is made.

In case of a raster type, the raster is loaded as a raster_image. In case of type :grid, the raster is also loaded as a raster_image, but on top of that, a grid is created containing the values (including transparency) of the raster image.
When the source image is a 24bit raster with an alpha channel containing the transparency of every pixel, the alpha channel is extracted. This is done with ImageMagick, stored in a file and its values loaded as a grid in the hidden property .alpha_channel.
