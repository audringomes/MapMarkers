Operating Mode

There are two operating modes, handled by the raster_symbol_plugin.
 

Goto draw

In transient_drawer mode, the plugin handles a :goto_request or :post_render_sets 's from the application databus. It uses the Goto Mappings to determine what and how to draw.

Draw raster symbols

In the draw_raster_symbols mode, a single raster symbol is drawn for a set of geometries. It offers the possibility to modify the drawing properties of the selected symbol.
See Drawing of symbols for more details.
