Retrieving symbols

These are the basic methods for getting a raster symbol with a certain name:

get_symbol()

*cb**gis cambridge_db_open* 


get_symbol()  IN  sw:raster_symbol_handler    B
 p_name

        ## This method returns an existing symbol with P_NAME or it
        ## will create and return a symbol from the symbol definition
        ## with P_PNAME

symbol()

*cb**gis cambridge_db_open* 


symbol()  IN  sw:raster_symbol_handler    B
 p_name

        ## Return the picture symbol with name P_NAME.
        ## If not found, return _unset

thumbnail_symbol()

*cb**gis cambridge_db_open* 


thumbnail_symbol()  IN  sw:raster_symbol_handler    B
 p_name

        ## Returns the thumbnail picture symbol for symbol with name
        ## P_NAME.
        ## If it does not exist, it is created with the default
        ## properties.
