Modules

 

This contents of this list is populated by the raster_symbol_handler. When a symbol is created from a module not yet in the list, the list is updated.
 

Lock
When the Lock symbol is pressed, the Symbol (definition) list lists only symbols from the selected module.
Otherwise the list shows all available raster symbols and raster Symbol Definitions.

Current module used in:

Copy When a Copy is initiated, and the module from the raster (definition) is different from the current module, the user is asked if he wants to copy the raster (definition) to the resources of the current module.
The associated raster will be copied to. The default name of the new symbol is "<NAME>(Copy)".
 
Add symbol When the button Add is pressed, the file dialog will open with the bitmap resource directory of the current module.
You can select any other folder, as long as it is within a Smallworld module.

Folder for loading configuration file.
